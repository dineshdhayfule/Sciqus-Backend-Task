# Student Course Management System - Frontend

This is the React frontend for the Student Course Management System.

## Prerequisites
- Node.js installed
- Backend server running on http://localhost:3000

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Features
- **Authentication**: Login for Students and Admins.
- **Admin Dashboard**: Manage students and courses.
- **Student View**: View available courses.

## Note
This frontend is designed to work with the provided Express backend. Ensure the backend is running before using the application.
